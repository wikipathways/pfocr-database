Summation of the proteomic values for the ERBB2− sample set for genes present in the regulation of actin cytoskeleton pathway for ERBB2− set from KEGG. The following notations were used: Red star: proteomics value >0 in ERBB2− set (red color is only selected for better clarity in the figure). Yellow box: proteomics value >3 spectral count only in ERBB2− tumor samples. Green boxes: potentially important pathway functions. Note that this is a different format for visual clarity as the red circles provide the best contrast with the green boxes used in the KEGG figures.
