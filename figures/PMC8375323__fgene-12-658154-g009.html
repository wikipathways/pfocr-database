TIMM17A Regulatory Network Contributing to Breast Cancer

Protein–protein interaction network of ATR kinase targets (GeneMANIA). Protein–protein interaction (PPI) network and functional analysis of the gene set enriched in targets of CDK1. Different colors of the network edge indicate the bioinformatics methods applied: co-expression, co-localization, physical interactions, pathway, predicted, genetic interactions, and shared protein domains. The different colors for the network nodes indicate the biological functions of the set of enrichment genes.
