Shiny GATOM: omics-based identification of regulated metabolic modules in atom transition networks

Metabolic module obtained for the example data of macrophages activated by IFNg + LPS. The KEGG network with atom topology was used. The module layout was done in Shiny GATOM. The shown pathway annotations were added separately.
