Canonical pathway analysis using MetaCore identified EGFR, PDGFR, and mTOR signalling as the top 3 canonical pathways modulated in FCD type II patients

Canonical pathway analysis using MetaCore identified EGFR, PDGFR, and mTOR signalling as the top 3 canonical pathways modulated in FCD type II patients. Red thermometers show an object that is upregulated. The big arrows indicate the “pathway start”. GR: group relation; CS: complex subunit; B: Boxes on lines denote the type of regulation where P is phosphorylation, B is binding and Z is catalysis.
