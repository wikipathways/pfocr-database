Core networks generated by the redGEM algorithm from iJN1411 genome-scale model

The core networks generated by the redGEM algorithm from iJN1411 genome-scale model. The core network was built around reactions (grey) that belong to the six subsystems of central carbon metabolism (glycolysis and gluconeogenesis, pentose phosphate pathway, pyruvate metabolism, TCA cycle and oxidative phosphorylation). Reactions belonging to one-reaction step, two-reaction-step, and three-reaction-step pairwise connections between the six subsystems are marked in red, cyan and magenta, respectively. The stoichiometry of the reduced models and a complete list of reactions and metabolites are provided in Additional file : File S2, Additional file : File S3 and Additional file : File S1
