Protein-protein interaction (PPI) network of DEGs

Protein-protein interaction (PPI) network of DEGs. Common DEGs PPI network constructed by STRING online database and Module analysis. a There were a total of 78 DEGs in the DEGs PPI network complex. The nodes meant proteins, the edges meant the interaction of proteins, blue circles meant downregulated DEGs, and red circles meant upregulated DEGs. b Module analysis via Cytoscape software (degree cutoff = 2, node score cutoff = 0.2, k-core = 2, and max. Depth = 100)
