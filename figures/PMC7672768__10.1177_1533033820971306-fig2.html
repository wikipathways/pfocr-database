Protein-protein interaction (PPI) network constructed using the STRING online database and Cytoscape software

The protein-protein interaction (PPI) network constructed using the STRING online database and Cytoscape software. A, The differentially expressed gene PPI network complex. The nodes denote proteins, and the edges denote the interaction of proteins. B-E, The clusters 1–4 for PPI were obtained by module analysis using the Cytoscape software (the degree of cutoff = 2, node score cutoff = 0.2, k-core = 2, and maximum depth = 100). The first 47 genes with high importance in the PPI network were screened out from B to E. They are IFI27, OASL, STAT1, IFI6, MMP1, MCM4, CHEK1, CCNE1, ISG15, RSAD2, TPX2, GBP1, TOP2A, KIF2C, PLAUR, SPP1, PLAU, CXCL12, CENPE, CDC20, MMP9, BST2, MKI67, CXCL8, CXCL1, IFI44, MMP13, IFI44 L, MMP3, TRIP13, COL14A1, COL7A1, COL1A2, COL4A5, COL5A2, COL4A2, ITGA6, COL4A6, COL4A1, TNC, APOL1, APP, SPARCL1, ADH1B, ALDH3A2, MAOB, and AOX1.
