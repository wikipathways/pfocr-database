Key downstream genes were identified in the PLK1 signaling pathway

Key downstream genes were identified in the PLK1 signaling pathway. Protein-protein interaction analysis was used to screen the important candidate genes regulated by PLK1 in bladder cancer cells by using STRING software (http://string-db.org). Analysis of the interaction between PLK1 and the differentially expressed genes about associated with cell proliferation signaling pathways by textmining (a), Experiments experiments (b), Database database (c) and, Coco-expression (d) and multiple methods (e). Analysis of the interaction between PLK1 and differentially expressed genes about associated with cell invasion and migration signaling pathways by textmining (f), Experiments experiments (g), Database database (h) and, Coco-expression (i) and multiple methods (j)
