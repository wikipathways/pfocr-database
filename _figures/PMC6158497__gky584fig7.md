  cycle and amino acid metabolisms. Histograms next to relevant gene names represent
  transcriptional change values from Table  (red boxes correspond to genes up-regulated

  to values from Table  (Hybrid-trap-seq data). Purple dots indicate CcpA-regulated
  genes.
