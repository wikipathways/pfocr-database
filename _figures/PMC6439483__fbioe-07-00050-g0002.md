caption: 'Squalene biosynthetic pathways and production strategies. (A) Squalene biosynthesis
  via MVA pathway in yeast, fungi, and algae. The engineering strategies for enhanced
  squalene production are as follows: overexpression of HMGR (Polakowski et al., ;
  Tokuhiro et al., ; Mantzouridou and Tsimidou, ; Dai et al., , ; Zhuang and Chappell,
  ; Rasool et al., ,; Kwak et al., ; Paramasivan and Mutturi, ; Han et al., ; Huang
  et al., ; Wei et al., ) and SQS (Dai et al., ; Zhuang and Chappell, ; Rasool et
  al., ,), downregulation of SQE (Garaiová et al., ; Hull et al., ; Zhuang and Chappell,
  ; Rasool et al., ,; Han et al., ) in yeast; downregulation of SQE in algae (Kajikawa
  et al., ). (B) Squalene biosynthesis via MEP pathway in bacteria. The engineering
  strategies for the enhanced squalene production are as follows: overexpression of
  DXS (Ghimire et al., ), IDI (Ghimire et al., ; Katabami et al., ) and FPS (Katabami
  et al., ), introduction of heterologous squalene producing enzyme(s) (Ghimire et
  al., ; Furubayashi et al., ; Katabami et al., ; Pan et al., ) in E. coli; overexpression
  of DXS and IDI, and the introduction of FPS into Synechococcus elongatus (Choi et
  al., ); deletion of SHC in Synechocystis (Englund et al., ) and Rhodopseudomonas
  palustris (Xu et al., ). MVA, mevalonate; MEP, 2-C-methyl-d-erythritol 4-phosphate;
  AACT, acetoacetyl-CoA thiolase; HMGS, 3-hydroxy-3-methylglutaryl-CoA synthase; HMGR,
  HMG-CoA reductase; DMAPP, dimethylallyl diphosphate; IDI, isopentenyl diphosphate
  isomerase; IPP, isopentenyl pyrophosphate; FPS, farnesyl pyrophosphate synthase;
  GPP, geranyl diphosphate; FPP, farnesyl pyrophosphate; SQS, squalene synthase; SQE,
  squalene epoxidase; GA3P,glyceraldehyde-3-phosphate; DXP, 1-deoxy-D-xylulose-5-phosphate;
  DXS, DXP synthase; DXR, DXP reductoisomerase; DLR, DXR-like enzyme; HMBPP, 4-hydroxy-3-methyl-but-2-enylpyrophosphate;
  IspH,IPP/DMAPP synthase; SHC, squalene hopene cyclase.'
