  a functional metagenomic screening method based on an EmrR-based biosensor for identifying
  lignin-degrading and transforming enzymes from metagenomic samples. A multicopper
  oxidase (CopA) enzyme capable of degrading lignin to vanillin and syringaldehyde
  was identified from a recovered metagenomic fosmid DNA library using this system
  []. b Transcriptional biosensors as screening methods to allow enzymatic characterization
