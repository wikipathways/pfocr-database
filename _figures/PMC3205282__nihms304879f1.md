caption: The hippocampus forms a unidirectional network with input arising from the
  entorhinal cortex that forms synaptic connections with the dentate gyrus (DG) via
  the perforant path (PP). Axons from DG project to CA3 pyramidal neurons via the
  mossy fibres pathway (MF). Axons from CA3 project to CA1 pyramidal neurons via the
  Schaeffer collateral pathway (SC). These CA1 neurons in turn send the main output
  back to the entorhinal cortex. CA3 and CA1 neurons can also receive input directly
  from the perforant path.
