  cells, iron-elevated ROS causes the oxidation and oligomerization of Tom20. Oxidized
  Tom20 induces Bax translocation to mitochondria, which facilitates cytochrome c
  release to cytosol. Once released, cytochrome c activates caspase-9, which then
  activates caspase-3. This caspase-3 activation further cleaves GSDME, and eventually
  triggers cell swelling and LDH release
