caption: 'Nrf2 response to oxidative stress. Under an oxidative environment, Keap1
  releases Nrf2, which translocates from the cytoplasm to the nucleus, where it binds
  to the ARE sequence, activating the anti-inflammatory response elements (ARE) pathway.
  This pathway allows for the transcription of genes that code for the main antioxidant
  enzymes, which will modulate mitochondrial damage by decreasing ROS and facilitating
  ATP synthesis. HO-1: heme oxygenase-1. NQO1: H quinone oxidoreductase-1. GST: glutathione
