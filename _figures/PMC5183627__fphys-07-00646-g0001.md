caption: TGR5 regulates different cell signaling pathways. TGR5 activates AKT (Kida
  et al., ; Perino et al., ), TRPA1 (Lieu et al., ), and Epac (Kumar et al., ) pathways.
  And it inhibits NF-κB (Pols et al., ; Wang et al., ; Yoneno et al., ), STAT3 (Guo
  et al., ), and RhoA/Rho kinase (Rajagopal et al., ) pathways. TGR5 has opposite
  functions in ERK1/2 pathway. In ciliated cholangiocytes, ERK1/2 is activated by
  TGR5 (Masyuk et al., ). But in non-ciliated cholangiocytes, TGR5 activation inhibits
  ERK1/2 (Masyuk et al., ).
